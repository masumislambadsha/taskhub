# Workspace Rules

## Expo HAS CHANGED
Read the exact versioned docs at https://docs.expo.dev/versions/v57.0.0/ before writing any code.

## Review + Test Before Reporting
Every time you make changes:
1. Review the edits (read the changed files to confirm correctness)
2. Run `npx eslint . --ext .ts,.tsx --quiet` and fix any errors
3. Run `npx tsc --noEmit` and fix any errors
4. Only then tell the user the changes are done
